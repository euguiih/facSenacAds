Responda as perguntas abaixo:
-Na tag input com atributo type="text", para que serve o atributo "placeholder"? O atributo placeholder é uma pequena frase ou palavra, que tem o intuito de ajudar usuários a entender como devem preencher determinado formulário, inserindo um texto contextual no campo a ser respondido.

-Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"? Para deixar uma "opção" já marcada.

-Na tag input para que serve o atributo "value"? Value serve para virar a 'etiqueta do botão', ou seja, o ue aparecerá em cima dele.

-Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez? Dentro de um fieldset, uma opção tipo rádio só é desmarcada quando outra opção RADIO com mesmo NAME(não é a mesma coisa que id) for selecionada.

-Para que serve a tag label? Para que serve o atributo "for" nessa tag? Um elemento HTML label representa uma legenda para um item em uma interface de usuário. Ele pode estar associado com um elemento de controle, colocando este dentro do elemento label , ou usando o atributo for que direciona a mesma função para um determinado 'id'.

-Na tag option para que serve o atributo "selected"? Serve para iniciar as opções com a tag tal qual esteja com este atributo. Ele irá mostrar a tag com essa opção primeiro e caso clique em cima, mostrará as outras opções.

-Nas tags de formulário para que serve o atributo "required"? Para indicar que a entrada do usuário é obrigatória em um elemento antes que um formulário possa ser enviado.

-Na tag form para que serve o atributo "action"?
-Na tag form para que serve o atributo "method"? A Tag possui dois parâmetros importantes: action e method. O primeiro serve para informar o arquivo que irá processar as informações enviadas pelo formulário. Já o segundo serve para informar o método de envio de informações.

-Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form? "post".

-Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos? "get"
